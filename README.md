🛡️ Shadow AI Detection System
Python 3.x  Cybersecurity  Machine Learning

A cybersecurity tool to detect unauthorized or risky usage of AI tools (ChatGPT, Claude, Gemini) within a network environment to prevent data leakage.

Features
✅ Simulated network traffic generation
✅ AI endpoint detection & classification
✅ Risk scoring and alerting
✅ Interactive visualizations


Quick Start
git clone https://github.com/username/Shadow-AI-Detection-System
cd Shadow-AI-Detection-System
pip install -r requirements.txt
